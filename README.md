# Ghislain Delabie's Personal Website

[![Live Site](https://img.shields.io/badge/visit-www.delabie.tech-blue)](https://www.delabie.tech)

Personal website and blog about AI engineering, data innovation, and sustainable mobility.

**Live at:** [www.delabie.tech](https://www.delabie.tech)

## About

This site serves as my professional portfolio and blog, featuring:
- Articles and analysis on AI, data science, and mobility
- Project portfolio showcasing work in AI engineering and sustainable transport
- Bilingual content (English/French) for broader accessibility
- Technical documentation and teaching materials

## Tech Stack

Built with [Jekyll](https://jekyllrb.com/) and deployed on [GitHub Pages](https://pages.github.com/).

### Theme & Credits

Based on the excellent [**al-folio**](https://github.com/alshedivat/al-folio) theme by [Maruan Al-Shedivat](https://github.com/alshedivat).
A clean, responsive Jekyll theme designed for academics and researchers.

### Key Plugins

- [**jekyll-polyglot**](https://github.com/untra/polyglot) - Multilingual support (EN/FR)
- [**jekyll-scholar**](https://github.com/inukshuk/jekyll-scholar) - Bibliography and citations
- [**jekyll-imagemagick**](https://github.com/zroger/jekyll-minimagick) - Responsive image generation
- [**jekyll-feed**](https://github.com/jekyll/jekyll-feed) - RSS/Atom feed generation

Full dependency list available in [`Gemfile`](Gemfile).

## Local Development

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve --livereload

# Visit http://localhost:4000
```
